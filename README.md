# Terabyte тАФ React App (JS)

Modern multi-page marketing site + services pages for **Terabyte**.

## Pages

- Home (`#/`)
- About (`#/about`)
- Services (`#/services`)
- Work (`#/work`)
- Contact (`#/contact`)

## Routing

This project uses **hash-based routing** (no extra dependencies), so routes work like `#/services`.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
